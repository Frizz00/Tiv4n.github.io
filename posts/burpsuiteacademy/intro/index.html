#  # 

In questa serie voglio approfondire l’accademy offerta da portswigger per imparare a conoscere, individuare e sfruttare le vulnerabilità web.

La Web Security Academy consiste in una vasta gamma di lezioni che comprendono la maggior parte delle vulnerabilità web presenti nelle applicazioni odierne. Le varie lezioni ti spiegano prima di cosa si tratta, successivamente ti spiegano come individuarla, sfruttarla ed eventualmente a imparare ad utilizzare alcuni metodi di evasione quando si cerca di mitigare la vulnerabilità senza risolverla (succede spesso). La cosa veramente utile è la possibilità di fare i vari laboratori per capire come funzionano le vulnerabilità dal punto di vista pratico. Anche se alcuni laboratori sono un po' complessi da capire all’inizio, penso che sia una risorsa estremamente utile che ti prepara in modo adeguato alle prime sfide nel tuo percorso lavorativo.

Se vuoi approfondire o farmi domande, invia una mail a: tiv4n@proton.me
